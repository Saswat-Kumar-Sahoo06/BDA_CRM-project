# BDA Management System

A full-stack **Role-Based Access Control (RBAC)** web application built using the MERN stack.  
It supports **Admin and BDA (Business Development Associate)** roles with authentication, protected routes, and dynamic dashboards.

---

## Features

### Authentication
- User Login & Registration
- JWT-based authentication
- Secure password hashing (bcrypt)
- Persistent login using localStorage

### Role-Based Access Control (RBAC)
- Admin role with full system access
- BDA role with limited permissions
- Dynamic UI rendering based on role
- Protected routes for security

### Dashboard
- Personalized welcome dashboard
- Role-based navigation sidebar
- Clean and responsive UI

### Security
- JWT token authentication
- Protected API routes
- Authorization middleware (backend)

---

## Tech Stack

### Frontend
- React.js
- React Router DOM
- Context API
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
